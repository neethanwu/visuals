# Visuals

A collection of agent skills for creating beautiful visual output — presentations, graphics, and more.

## Skills

### slides

Builds stunning single-file HTML presentations with 37 curated design styles, interactive effects, and smart style inference. Give it content (or just a topic) and it builds a complete deck. One file, any browser, zero dependencies.

```bash
npx skills add neethanwu/visuals --skill slides
```

```
/slides My startup pitch for an AI-powered design tool
```

**37 design styles** — from Swiss typography to synthwave to glassmorphism. The skill matches your content to the best style automatically.

**49 font trios** — curated heading + body + mono combinations via Google Fonts.

**Interactive effects** — entrance animations, WebGL shaders, GSAP stagger, style-specific effects (perspective grids, halftone dots, scan lines, gradient mesh).

**Data visualization** — Chart.js, D3.js, styled tables and KPI cards, all themed to match the deck.

More skills coming — social graphics, posters, on-brand.
