# Swiss Clean Expressive

## Visual Direction
Swiss grid precision amplified by controlled expressiveness. The base is a white canvas with a near-black sidebar, a system of numbered navigation (01, 02, 03...), and a vivid iOS-standard red (#FF3B30) that lands with precise, almost surgical impact. The typography pairs Space Grotesk (structural titles and data) with Inter (body and labels), maintaining clear functional zones. Strategic dark inversions — hero metric cards flipped to near-black, table headers rendered as solid dark bars — create bold value contrasts that direct the eye without chaos. Page titles run at 54px with -2px tracking, creating a confident, editorial authority. Zero corner radius throughout enforces Swiss precision. This is organized boldness: every expressive choice is systematically placed, not spontaneous.

## Color Palette

| Role | Hex | Description |
|------|-----|-------------|
| Background | #FFFFFF | Pure white — primary slide canvas |
| Sidebar / Dark Zone | #0A0A0A | Near-black for sidebar, inverted hero cards, table headers |
| Text Primary | #0A0A0A | Matches sidebar; typographic system and structure read as one |
| Text Secondary | #777777 | Mid-gray for supporting copy, metadata, captions |
| Accent | #FF3B30 | iOS standard red — vivid, precise, used for the single focal element |
| Success | #22C55E | Bright green — positive indicators and growth metrics |
| Border | #E0E0E0 | Light gray structural borders on white surface |

## Typography

### Display Font
- **Family:** Space Grotesk
- **Weights:** 700 (slide titles, hero metrics, featured callouts), 600 (section headings, card titles)
- **Size Range:** 48–64px for slide titles, 28–36px for section headings, 20–24px for card titles
- **Letter Spacing:** -2px on display titles (48px+); standard tracking on headings; +2px on logo/wordmark
- **Style Notes:** Space Grotesk 700 at -2px tracking creates a tightly-packed, high-authority title. Navigation numbers (01, 02, 03...) are set in Space Grotesk 600 at the same scale as section headings, creating a numbered editorial cadence.

### Body Font
- **Family:** Inter
- **Weights:** 400 (body copy, descriptions), 500 (labels, table content, metadata)
- **Size Range:** 13–16px body (400), 11–13px labels (500)
- **Line Height:** 1.6 for body paragraphs, 1.35 for label/metadata stacks

### Data/Mono Font (if applicable)
- **Family:** Space Grotesk
- **Weights:** 700 (primary KPIs), 600 (secondary metrics)
- **Usage:** All numerical KPIs and large data figures. Set in #0A0A0A or #FFFFFF depending on card inversion. Apply iOS red (#FF3B30) to the single most critical metric per dark-inverted hero card. -2px tracking on figures 40px and above.

## Design Principles
1. Numbered navigation is structural — slide sections are anchored by two-digit numbers (01, 02...) in Space Grotesk, creating a Swiss-editorial page reference system across the deck.
2. Dark inversions are systematic — hero metric cards and table headers use #0A0A0A fill with #FFFFFF text as a deliberate, structured pattern, not random emphasis.
3. Red is precision ammunition — #FF3B30 appears once per slide, on the single most important element (primary metric, key callout, CTA). Its vividness on white is extreme; use it with precision.
4. Zero radius enforces Swiss discipline — every card, tag, button, and container is square-cornered. The only softness in the system comes from the type itself.
5. Space Grotesk leads, Inter supports — the functional division is clear: Space Grotesk owns titles, metrics, and structural numbers; Inter owns body copy, labels, and supporting data.

## Slide Application Notes
- **Title Slides:** White canvas, Space Grotesk 700 title at 54–64px (-2px tracking) in #0A0A0A. Navigation number (01 or the presentation chapter number) in Space Grotesk 600 #777777 at top-left. Red (#FF3B30) underline rule beneath the title or on one accent element.
- **Content Slides:** White canvas, #0A0A0A sidebar zone (if two-column) or full-width with section number. Dark-inverted hero metric card (#0A0A0A bg, white text, red primary figure). Border #E0E0E0 on supporting cards.
- **Data/Chart Slides:** Dark-inverted table headers (#0A0A0A rows, white Space Grotesk 600 column labels). Primary chart series in #0A0A0A; success/growth series in #22C55E; featured series in #FF3B30. Space Grotesk 700 for all large metric callouts.
- **Closing Slides:** White canvas or full dark inversion depending on deck tone. Space Grotesk 700 closing statement at 48–54px (-2px). Red on CTA element. Logo/wordmark in Space Grotesk with +2px tracking.
- **Do:** Commit to the numbered navigation system (01, 02...) as a consistent anchor across every section slide. Use dark inversions for hero cards and table headers as a systematic pattern.
- **Don't:** Apply red to more than one element per slide. Add any corner radius. Use Inter for titles or large metric figures — Space Grotesk owns those roles.
