# Font Trio Pairing Guide

Source: [fonttrio.xyz/pairings](https://www.fonttrio.xyz/pairings)

Each trio includes a **Heading** font, **Body** font, and **Code/Mono** font.
Install via CLI: `npx shadcn@latest add @fonttrio/[name]`

## Geist Font Family Reference

The Geist family by Vercel. Available on Google Fonts (Sans + Mono). Pixel is not yet on Google Fonts — use CDN from GitHub releases if needed.

### Geist Sans
- **Weights:** Thin (100), Ultra Light (200), Light (300), Regular (400), Medium (500), Semi Bold (600), Bold (700), Black (800), Ultra Black (900)
- **Google Fonts:** `https://fonts.googleapis.com/css2?family=Geist:wght@100..900&display=swap`
- **Character:** Clean geometric sans, Vercel's signature typeface. Excellent for product/tech slides, minimal corporate, and modern clean aesthetics.

### Geist Mono
- **Weights:** Thin (100) through Ultra Black (900) — full range
- **Google Fonts:** `https://fonts.googleapis.com/css2?family=Geist+Mono:wght@100..900&display=swap`
- **Character:** Monospace companion to Geist Sans. Ideal for code slides, data labels, metrics, and technical content.

### Geist Pixel
- **Variants:** Square, Grid, Circle, Triangle, Line — five geometric sub-variants
- **Google Fonts:** Not available yet. Use hosted files from [GitHub releases](https://github.com/vercel/geist-font/releases)
- **Character:** Experimental pixel/geometric display face. Use for creative, futuristic, or playful title slides only — not for body text. Works well at very large sizes (80px+) as a visual statement.
- **Note:** For single-file HTML slides, self-host the WOFF2 file or use a CDN link to the GitHub release. Cannot be loaded via Google Fonts `<link>` tags yet.
